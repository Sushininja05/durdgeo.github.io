I'm Durdge-o! I'm an amateur game developer, programmer, artist, etc. I wear many hats but it's mainly just to fuel my hobbies, what I know I've picked up for funsies. I'm a big fan of source engine, fighting games, rhythm games, freeware, kusoge, and the like. If I'm not making something, I'm playing something.

On that note, this website was created so I had a central hub for all my hobbies. It started as a project to fight my boredom but quicky ballooned as time went on. Despite most of this site just being rudimentary html examples, I hope this website inspires others to start their own websites.
